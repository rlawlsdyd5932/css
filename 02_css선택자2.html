동위 선택자

동위 관게(형제 관계)에서 뒤에 위치한 태그를 선택할 때 사용한다.

바로 뒤의 요소 선택 시: 선택자a+선택자b {설정내용}으로 사용하며,
        뒤에 있는 몯두를 선택할 때는 ,선택자a~선택자b {설정내용}으로 사용된다

반응 선택자

사용자의 움직임에 따라 달라지는 선택자

사용자가 클릭한 태그: 선택자:active{설정내용},
        사용자의 마우스가 올라가 있는 태그: 선택자:hoover{설정내용}으로 사용된다.
active 테스트
hover 테스트

상태 선택자

입력 양식의 상태에 따라 선택되는 선택자

체크 상태의 input태그 선택: input 태그:chedked{설정내용}

초점이 맞추어진 input태그 선택자



사용 가능한 input태그 선택자

태그: enabled{설정내용}, 태그:disabled{설정내용}
